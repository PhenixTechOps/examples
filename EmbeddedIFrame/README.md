# Embeded IFrame example example

## Description

Code example on how to embed a reactive iframe using a PhenixRTS permalink.

## Getting Started

### Dependencies

There are no outside dependencies.

## How to use

- Run on a local webserver
- Run on JSFiddle.

# Instructions

1. Publish a stream using the customer portal.
2. Generate viewing permalink in the Customer portal under the edgeAuth tab.
3. Copy the permalink url, open the index.html file in a text editor.
4. Replace `PERMALINK` in the src value and save the file.
5. Open index.html in a browser.
